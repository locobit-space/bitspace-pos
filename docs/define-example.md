## 4. ระบบคำนวณอัตโนมัติ (Auto Calculation)

### วิธีทำ (ใน Google Sheets)

1. **Sheet 1: สูตรเมนู (Recipes)**  
   - ป้อนส่วนผสม + ปริมาณ + ราคา → คำนวณต้นทุนอัตโนมัติ

2. **Sheet 2: สต๊อก (Stock)**  
   - เมื่อขายเค้ก 1 ปอนด์ → ดึงข้อมูลจากสูตร → หักสต๊อกอัตโนมัติ

3. **Sheet 3: รายงานกำไร (Profit Report)**  
   - ยอดขาย - ต้นทุน = กำไรสุทธิ

---

### ตัวอย่างสูตรใน Google Sheets

```excel
=SUMPRODUCT(ปริมาณ, ราคาต่อหน่วย)   → ต้นทุนรวม
=IF(สต๊อก < 5, "เตือนสั่งซื้อ!", "พอใช้")
```
